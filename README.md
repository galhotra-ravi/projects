## 🗺️ Rajasthan Map Quiz

🔗 **Live Demo:** https://raj-map-quiz.vercel.app/

An interactive web application to practice Rajasthan district identification. Designed for students preparing for competitive exams and geography-based assessments.

### ✨ Features

- Interactive SVG-based Rajasthan map
- District identification practice
- Clean and responsive UI
- Fast and lightweight performance
- Useful for map-filling exams

### 🎯 Use Cases

- Competitive exam preparation (SSC, State Exams)
- Rajasthan GK practice
- School/college geography learning

### 🛠️ Tech Stack

- Reactjs
- SVG (Map Handling)
- Vercel (Deployment)

### 💡 What This Project Demonstrates

- SVG interaction handling (hover, click)
- Dynamic UI updates
- Practical educational tool design
- Clean frontend architecture
